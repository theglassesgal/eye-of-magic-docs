how the bridge works

this system is a real-time two-way bridge between discord and roblox. when you run a command in discord, the python bot logs it in a queue. your live roblox server constantly pings the bot's web server every few seconds. once roblox sees a pending command, it pulls it down and executes the moderation action in-game immediately.

moderation commands

mg.ban <username> [reason]

permanently bans a user globally from the entire game universe.

mg.unban <username>

lifts a global universe ban so the user can rejoin.

mg.kick <username> [reason]

forcefully removes a user from their active server instance.

mg.flag <username> [reason]

marks a user in the moderation database. when they join a server, their log will be highlighted.

mg.unflag <username>

removes the database warning flag from a user.

utility commands

mg.clear [amount]

purges discord messages. leaving it blank or doing over 100 clones and deletes the channel for an instant wipe.

mg.help

shows the quick reference menu in discord.

host console commands

these are typed directly into your computer terminal, not discord.

stop

cleanly shuts down the web server and takes the bot offline.

st

prints the status type reference sheet to your terminal.

status <type> <message>

updates the bot's custom discord activity status.
