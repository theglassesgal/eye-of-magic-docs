Server Integration

Deploy the Eye of MAGIC bridge to your own community.

[ + ] ADD BOT TO DISCORD SERVER

Setup Instructions:

  • Invite the bot using the portal above.
  • In your Discord server, run mg.setlog #channel-name to bind your logging feed.
  • Run mg.genscript to generate a unique, authenticated bridge script.
  • Place the generated script into ServerScriptService inside Roblox Studio.

Architecture Overview

This system operates as a real-time bridge linking Discord directly to active Roblox server instances through a centralized host application queue.

Moderation Directives

mg.ban <username> [reason]

Issues a persistent universe ban.

mg.kick <username> [reason]

Forcefully terminates the user's connection.

mg.unban <username>

Purges the target user from the global blacklist.

mg.flag <username> [reason]

Applies an administrative warning tag.

mg.userinfo <username>

Retrieves profile records and historical data directly from Roblox.

System Utilities

mg.clear [amount]

Purges channel message logs using automated recreation.

mg.help

Retrieves the localized command reference index.

Host Terminal Directives

These commands are strictly for the local host terminal.

stop

Initiates a safe shutdown sequence.

st

Outputs the valid presence status codes.

status <type> <message>

Modifies the rich presence activity string.

mg.maintenancemode <true/false>

Toggles internal maintenance state across the web layout and app profiles.
