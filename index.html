Eye of MAGIC

Architecture Overview

This system operates as a real-time bridge linking Discord directly to active Roblox server instances through a centralized host application queue.

Moderation Directives

mg.ban <username> [reason]

Issues a persistent universe ban.

mg.kick <username> [reason]

Forcefully terminates the user's connection.

System Utilities

mg.clear [amount]

Purges channel message logs using automated recreation.

Host Terminal Directives

status <type> <message>

Modifies the rich presence activity string.
