# Command Reference

Welcome to the official command reference for Eye of MAGIC. 
This guide covers both the in-game Discord moderation commands and the backend console commands used by the bot host.

---

## Moderation Commands
All standard moderation commands use the mg. prefix and must be run inside Discord by authorized personnel.

### Ban
Bans a user globally across the entire game universe.
```text
mg.ban <username> [reason]
